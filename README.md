# Global 10Y Yields

A live GitHub Pages dashboard focused on sovereign 10-year government bond yields across countries and regions.

## What this tool does

This dashboard provides a clean overview of global 10-year sovereign yields.

It shows:

- countries tracked
- highest 10Y yield
- lowest 10Y yield
- average 10Y yield
- median 10Y yield
- benchmark 10Y markets
- 10Y daily movers
- regional 10Y yield averages
- lowest 10Y yield snapshot
- full live 10Y table
